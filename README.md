## 产品概述

### 基础介绍
高级文本管理工具是一款功能全面的桌面应用程序，专为高效管理各类文本内容而设计。它支持纯文本、Markdown和HTML格式，提供强大的分类、标签和搜索功能，帮助用户轻松组织和管理大量文本资料。

### 适用场景
- **学生**：整理课堂笔记、研究资料
- **作家**：管理创作素材、草稿和灵感
- **程序员**：保存代码片段和技术文档
- **研究人员**：收集和分类研究资料
- **商务人士**：管理会议记录、商务文档
- **知识工作者**：构建个人知识库

## 详细使用指南

### 安装与启动
1. 下载最新版本的可执行文件
2. 双击运行程序
3. 首次启动会自动创建数据库文件

*小贴士：程序会在同目录下创建`text_manager_enhanced.db`文件存储所有数据*

### 基础操作
#### 创建新文本
1. 点击工具栏"新建"按钮(或按Ctrl+N)
2. 输入标题和内容
3. 选择适当的格式(纯文本/Markdown/HTML)
4. 点击"保存"按钮(或按Ctrl+S)

*示例：创建一篇Markdown格式的笔记，使用#标记标题，*标记列表项*

#### 分类管理
1. 左侧面板点击分类树
2. 右键菜单可添加/编辑/删除分类
3. 拖拽分类可调整层级关系

*示例：创建"工作"-"项目A"-"会议记录"多级分类结构*

#### 标签使用
1. 在编辑界面"标签"输入框输入标签
2. 多个标签用逗号分隔
3. 通过左侧标签云筛选文本

*示例：为技术笔记添加"Python, 算法, 优化"等标签*

### 高级操作
#### 批量导出
1. 选择多个文本(按住Ctrl多选)
2. 点击"批量操作"按钮
3. 选择导出格式和目录
4. 点击"导出选中项"

*专业提示：批量导出适合定期备份或迁移数据*

#### 文本分析
1. 打开要分析的文本
2. 点击"文本分析"按钮
3. 查看各项统计数据和图表
4. 分析相似文本获取关联内容

*示例：分析文章可读性，调整句子长度使评分达到70+*

#### 数据库维护
1. 定期使用"工具"-"优化数据库"
2. 自动备份功能默认开启
3. 手动备份可通过复制数据库文件

*技术说明：程序使用SQLite数据库，VACUUM命令可优化存储空间*

## 专业功能解析

### 全文搜索技术
本工具实现了两种搜索模式：
1. **普通搜索**：基于LIKE的模糊匹配，支持拼音首字母
2. **全文检索**：使用SQLite FTS5引擎，支持词干提取和高级查询语法

*技术细节：FTS5使用porter词干分析器，支持英文词形还原*

### 相似文本算法
相似度计算基于多维特征：
1. 词汇特征(词数、独特词数、词汇多样性)
2. 语言特征(中英文比例)
3. 结构特征(平均句长、段落数)
4. 内容特征(问句比例、感叹句比例)
5. 关键词特征(TF-IDF加权)

*算法公式：相似度 = 0.6×数值特征相似度 + 0.4×关键词相似度*

### 自动备份系统
智能备份策略包含：
1. **循环清理**：保留最多30个备份
2. **WAL模式**：确保备份时数据一致性
3. **定时触发**：程序关闭时自动执行

*最佳实践：建议重要数据额外手动备份到云存储*
