## 产品概述

**高级文本管理工具**是一款功能全面的跨平台文本信息管理软件，集文本编辑、分类管理、标签系统、全文检索、数据分析于一体。软件采用SQLite数据库存储，支持Markdown和HTML格式，提供即见即所得(WYSIWYG)编辑体验。

**发布日期**: 2025-05-14

## 详细使用指南

### 安装与启动
1. 下载对应系统的可执行文件
2. 首次启动自动创建数据库文件(`text_manager_enhanced.db`)
3. 主界面分为左侧导航区和右侧编辑区

### 基础操作
#### 创建新文本
1. 点击"新建"按钮(或按Ctrl+N)
2. 输入标题和内容
3. 选择分类和标签(可选)
4. 点击"保存"(或按Ctrl+S)

*示例*: 学生创建课堂笔记:
- 标题: "文学史-宋代诗词特点"
- 分类: "学习/文学"
- 标签: "文学,宋代,课堂笔记"

#### 搜索文本
1. 在搜索框输入关键词
2. 支持以下搜索方式:
   - 普通搜索: 标题/内容关键词
   - 拼音搜索: 输入"wxs"可匹配"文学史"
   - 高级搜索: 按日期/字数范围筛选

### 分类管理
1. **添加分类**: 通过"分类"菜单创建
2. **嵌套分类**: 拖拽分类可创建层级结构
3. **分类颜色**: 右键分类可设置标识色

*专业提示*: 使用颜色编码可以建立视觉分类系统，例如:
- 红色: 紧急/重要文档
- 蓝色: 参考资料
- 绿色: 已完成项目

### 标签系统
1. **添加标签**: 在编辑区标签输入框用逗号分隔
2. **标签云**: 左侧面板显示所有标签
3. **智能标签**: 通过"自动添加标签"提取关键词

*企业应用示例*:
- 项目文档标记: "2025Q3,市场分析,机密"
- 会议记录标记: "2025-05,董事会,决议"

### 格式支持
| 格式类型 | 适用场景 | 特点 |
|---------|---------|------|
| 纯文本  | 简单笔记/代码 | 无格式干扰 |
| Markdown | 技术文档/博客 | 轻量级标记 |
| HTML | 网页内容/富文本 | 完整格式支持 |

### 数据导入导出
1. **导入**: 支持.txt/.md文件
2. **导出**: 支持.txt/.md/.html格式
3. **批量导出**: 可同时导出多个文档

*作家工作流示例*:
1. 在软件中完成初稿(Markdown格式)
2. 添加"初稿,小说,第3章"标签
3. 导出为HTML发送给编辑审阅

### 回收站管理
1. 删除的文本会进入回收站
2. 可恢复或永久删除
3. 回收站支持搜索功能

## 专业应用场景

### 软件开发
1. **代码片段管理**:
   - 分类: "Python/Web开发"
   - 标签: "Django,ORM,代码片段"
   - 使用纯文本模式保持代码格式

2. **API文档**:
   - 使用Markdown编写
   - 添加"API,v2.1,未完成"标签
   - 通过全文搜索快速定位接口说明

### 学术研究
1. **文献管理**:
   - 每篇文献创建独立条目
   - 添加作者、年份等标签
   - 使用分类建立学科体系

2. **实验记录**:
   - 日期分类(2025/05)
   - 标签标记实验类型
   - 导出为HTML生成报告

### 内容创作
1. **多平台发布**:
   - 在软件中完成内容创作
   - 使用Markdown保持格式纯净
   - 导出为各平台所需格式

2. **协作编辑**:
   - 导出HTML发送给团队成员
   - 通过标签系统跟踪修改版本
   - 使用分类管理不同阶段内容

### 企业知识库
1. **部门知识共享**:
   - 分类按部门组织
   - 标签标记知识类型
   - 设置权限分类(公开/部门/机密)

2. **流程文档**:
   - 使用HTML格式保留复杂格式
   - 添加"SOP,2025版,财务"标签
   - 通过全文搜索快速检索流程
