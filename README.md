## 产品概述

**高级文本管理工具**是一款功能强大的跨平台文本编辑与管理软件，专为高效处理各类文本内容而设计。它支持纯文本、Markdown和HTML三种格式，提供从基础编辑到高级分析的全套解决方案。

### 基础版特点（适合普通用户）
- 简洁直观的界面设计
- 多格式文本编辑（纯文本/Markdown/HTML）
- 分类与标签管理
- 回收站功能防止误删
- 自动保存与备份

### 专业版特点（适合高级用户）
- 全文检索与高级搜索
- 文本分析与统计
- 批量操作功能
- 数据库优化与维护
- 自定义快捷键
- 智能颜色编码系统

## 核心功能

### 1. 文本编辑
- **三种编辑模式**：
  - 纯文本：适合简单笔记
  - Markdown：适合结构化写作
  - 即见即所得：适合富文本内容

*示例*：撰写技术博客时，可先用Markdown写作，再切换到HTML模式添加复杂格式。

### 2. 分类与标签
- **树形分类**：无限层级分类结构
- **智能标签**：自动颜色编码，支持批量管理

*操作指南*：右键分类可设置颜色，相同颜色项目自动归类。

### 3. 搜索功能
- **基础搜索**：标题/内容/拼音首字母搜索
- **高级搜索**：按日期、字数、类型筛选
- **历史记录**：自动保存最近10次搜索

*专业技巧*：使用"全文检索"模式进行更精确的内容搜索。

### 4. 回收站
- 安全删除机制
- 恢复功能
- 永久删除选项

*注意*：回收站中项目30天后自动清理，重要内容请及时恢复。

### 5. 批量操作
- 批量修改分类
- 批量添加标签
- 批量导出文档

*效率技巧*：Ctrl+点击可多选文件进行批量操作。

## 详细使用指南

### 安装与启动
1. 下载最新版本（支持Windows/macOS/Linux）
2. 首次运行自动创建数据库
3. 界面布局介绍：
   - 左侧：分类树和文件列表
   - 右侧：编辑区域和预览

*注意*：建议定期备份`text_manager_enhanced.db`文件。

### 基础工作流程
1. 新建文本（Ctrl+N）
2. 选择格式（纯文本/Markdown/HTML）
3. 编辑内容
4. 添加分类和标签
5. 保存（Ctrl+S）

*示例*：创建"项目计划"文档：
- 分类：工作/项目A
- 标签：重要, 待审核
- 格式：Markdown

### 高级功能详解

#### 文本分析
- **基本统计**：字数、字符类型分布
- **可读性评分**：评估文本难度
- **情感分析**：检测文本情感倾向
- **关键词提取**：自动识别重要词汇

*专业解读*：可读性评分基于Flesch-Kincaid公式，60分以上适合大众阅读。

#### 相似文本查找
使用多维特征匹配算法：
1. 词汇特征
2. 结构特征
3. 内容特征
4. 关键词特征

*应用场景*：查找重复内容或相关笔记。

#### 数据库维护
- 定期优化（工具>优化数据库）
- 自动备份（每天首次启动时）
- 手动备份（文件>备份）

*技术细节*：使用SQLite WAL模式确保数据安全。

## 专业功能介绍

### 1. 智能颜色编码系统
基于斐波那契散列算法生成和谐颜色：
```python
def generate_harmonious_color(item_id):
    golden_ratio = 0.618033988749895
    hue = (item_id * golden_ratio) % 1.0
    return QColor.fromHsvF(hue, 0.6, 0.9)
```

*设计理念*：确保颜色既美观又具备足够区分度。

### 2. 全文检索实现
使用SQLite FTS5扩展：
```sql
CREATE VIRTUAL TABLE texts_fts USING fts5(
    title, content, 
    tokenize="porter unicode61"
)
```

*性能优化*：porter词干提取器提升搜索效率。

### 3. 自动备份机制
- 循环备份（保留最近30个版本）
- WAL模式确保备份一致性
- 智能清理旧备份

*技术细节*：备份文件存储在程序目录下的backups文件夹。
