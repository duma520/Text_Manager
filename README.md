## 产品概述

**高级文本管理工具**是一款功能全面的跨平台文本编辑与管理软件，集成了文本编辑、分类管理、标签系统、全文搜索、数据分析等多项功能。采用PyQt5框架开发，支持Windows、macOS和Linux系统。

### 核心特点
- **多格式支持**：纯文本、Markdown、富文本(HTML)三种编辑模式
- **智能管理**：分类树、标签云、高级搜索功能
- **数据分析**：文本统计、关键词提取、相似度分析
- **安全可靠**：回收站机制、自动保存、数据库备份

## 基础功能介绍

### 1. 文本编辑
**三种编辑模式自由切换**：
- **纯文本模式**：适合快速记录
  *示例：购物清单、简单备忘*
  
- **Markdown模式**：支持语法高亮
  ```markdown
  # 标题
  - 列表项
  **加粗**文本
  ```
  
- **即见即所得模式**：可视化排版
  *适合需要复杂排版的文档*

### 2. 分类管理
- 树形分类结构，支持无限层级
- 拖拽调整分类顺序
- 颜色标记重要分类

### 3. 标签系统
- 为文本添加多个标签
- 通过标签快速筛选
- *示例：为"项目报告"添加"重要"、"财务"等标签*

### 4. 搜索功能
- 普通搜索：标题/内容关键词
- 拼音首字母搜索：输入"wj"可找到"文件"
- 高级搜索：按日期、字数等条件筛选

## 专业功能详解

### 1. 文本分析
**功能入口**：点击"文本分析"按钮

**核心指标**：
- **基本统计**：字数、中英文比例、段落数
- **可读性评分**：评估文本阅读难度
- **情感分析**：识别文本情感倾向
- **关键词提取**：自动提取核心词汇

**专业应用**：
- *市场分析*：从客户反馈中提取情感倾向
- *学术研究*：分析文献关键词分布

### 2. 相似文本查找
- 基于多维特征计算相似度
- 显示相似度评分和关键特征
- *应用场景：查重检测、相关文档推荐*

### 3. 批量操作
- 批量修改分类
- 批量添加标签
- 批量导出文档
- *企业应用：一次性处理数百份文档*

### 4. 数据库管理
- 数据库优化(VACUUM)
- 定期备份机制
- 版本升级支持
